# Aplicação de Mensagem 

Foi implementado uma aplicação de mensagem de grupo e de mensagem direta utilizando RPyC (Remote Python Calls) e ZeroMQ. 
## Pre requisitos 

- Python 3.8 ou acima 
- RPyC library
- ZeroMQ library
